# Trazabilidad Real - Bootstrap

Semilla minima para que un proyecto nuevo no nazca sin capa verificable de trazabilidad.

## Estado esperado al arrancar

- `nodes.json` puede empezar vacio si el estado es `bootstrap`.
- `edges.json` puede empezar vacio si el estado es `bootstrap`.
- `coverage.json` deja claro que la cobertura todavia no esta medida.

## Regla

La plantilla solo entrega la estructura. El proyecto debe rellenar relaciones reales en cuanto existan memoria, anejos, Excel o BC3 vigentes.
