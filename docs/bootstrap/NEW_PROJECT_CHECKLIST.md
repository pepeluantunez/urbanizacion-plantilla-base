# New Project Checklist

1. Crear repo GitHub privado del expediente.
2. Clonar la plantilla base.
3. Rellenar `CONFIG/proyecto.template.json`.
4. Ejecutar `scripts/iniciar_proyecto_estandar.ps1`.
5. Revisar `AGENTS.md`, `.github/` y `CONFIG/trazabilidad_profiles.json`.
6. Confirmar `MANIFEST_VIGENCIA.md` inicial en `DOCS/` y `PRESUPUESTO/`.
7. Hacer primer commit de bootstrap.
