# Migration Acceptance

La plantilla se considera lista cuando:

1. `scripts/iniciar_proyecto_estandar.ps1` arranca un proyecto limpio.
2. Todos los tokens de personalizacion se sustituyen.
3. `.github/`, `AGENTS.md` y `CONFIG/` quedan operativos.
4. No aparecen rutas del proyecto origen.
5. El nuevo repo arranca con primer commit claro de bootstrap.
